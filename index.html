01

Инструменты

Программы для обхода DPI-фильтрации. Все бесплатные, с открытым кодом. Работают на Windows, Linux и Android.

⊞

GoodbyeDPI

Обход DPI на Windows. Убирает блокировку на уровне системы, не является VPN — просто модифицирует пакеты.

github.com/ValdikSS/GoodbyeDPI →
⊟

zapret

Аналог GoodbyeDPI для Linux и роутеров. Поддерживает nftables и iptables, много готовых пресетов для разных провайдеров.

github.com/bol-van/zapret →
⊡

ByeDPI

Для Android. Обход DPI без root-прав, без рекламы и без телеметрии. Минимальный размер и энергопотребление.

github.com/dovecote/ByeDPI →
02

VPN-клиенты

Приложения для подключения к VPN. Установите, импортируйте конфиг — и вы в сети.

⊞

v2rayN

Для Windows. Самый популярный клиент для VLESS/Xray. Удобный интерфейс, поддержка подписок и QR-кодов.

github.com/2dust/v2rayN →
⊟

v2rayNG

Для Android. Аналог v2rayN на мобильные. Поддержка всех протоколов, импорт из ссылки или QR.

github.com/2dust/v2rayNG →
⊡

Nekobox

Для Windows/Linux/macOS. Кросс-платформенный клиент на базе Qt. Поддержка Xray и sing-box.

github.com/qr243vbi/nekobox →
⊡

Hiddify

Для iOS. Клиент с красивым интерфейсом, поддержка VLESS Reality. Есть в App Store.

App Store →
⊡

Streisand

Для iOS/macOS. Open source клиент, поддержка VLESS, Trojan, Shadowsocks. Есть в App Store.

App Store →
⊡

v2rayA

Веб-интерфейс для управления V2Ray. Установите на сервер и управляйте через браузер.

github.com/v2rayA/v2rayA →
⊡

Happ

Для Windows/macOS/Linux. Многофункциональный VPN-клиент с поддержкой множества протоколов. Удобный интерфейс и автоматическая настройка.

happ.su →
⊡

V2Box

Для Android. Простой клиент с поддержкой VLESS и подписок. Доступен в Google Play для РФ.

Google Play →
